# Sample gym PT Link Project

이 프로젝트는 Sample gym의 트레이너 소개, PT 프로그램 안내, 카카오 상담 예약 연결을 위한 모바일 우선 링크 페이지입니다.

## 프로젝트 개요

- 목적: 인스타그램 프로필 링크에서 바로 접속 가능한 PT 안내/예약 페이지 제공
- 핵심 기능: 트레이너 프로필, 프로그램 가격 안내, 카카오 상담 예약 CTA, 인스타 링크 연결
- 타겟 디바이스: 모바일 우선

## 기술 스택

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- npm
- Vercel

## 저작권 및 사용 제한 (중요)

본 저장소의 모든 소스코드, 디자인, 문구, 이미지 및 기타 산출물의 저작권은 프로젝트 소유자에게 있습니다.

- 무단 복제, 무단 배포, 무단 수정, 무단 재사용을 금지합니다.
- 상업적 이용 및 2차 저작물 작성은 사전 서면 허가 없이 불가합니다.
- 본 프로젝트의 구조/표현/콘텐츠를 그대로 또는 유사하게 전재하는 행위를 금지합니다.
- 위반 시 민형사상 법적 조치를 포함한 권리 보호 절차를 진행할 수 있습니다.
